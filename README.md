## Business Environment Analysis in European Countries

This project analyzes the business environment across European countries using statistical methods in R.  
The aim of the study is to understand the main economic and digital differences between countries and to identify groups of countries with similar characteristics.

## Project Activities

- Data cleaning and preprocessing  
- Outlier detection and removal  
- Descriptive statistical analysis  
- Correlation analysis  
- Principal Component Analysis (PCA)  
- Cluster analysis to group countries with similar profiles  
- Regression analysis using the extracted components  
- Data visualization and geographic representation of results  

## Tools and Technologies

- R programming language  
- ggplot2  
- FactoMineR  
- corrplot  
- cluster  
- plotly  


Through this project, I developed practical skills in data analysis, multivariate statistical methods and data visualization.
